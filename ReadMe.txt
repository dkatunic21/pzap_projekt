ReadMe

Preuzmite zip datoteku sa svim programskim rješenjima i datotekama.

Zatim odite u main.py funkciju i pokrenite je. Kada se program pokrene kliknite na link koji se nudi u konzoli.

!!!OPREZ: Nemojte pokretati ostale programske datoteke jer su one služile samo jednom, za pohranu podataka u bazu. Ako se opet pokrenu, podaci će se poduplati i aplikacija neće kvalitetno prikazivati podatke.
